Rovas Tools APT Repository

Packages for installing and updating the Rovas Linux tool set on Ubuntu, Xubuntu, Linux Mint, and compatible Debian-based systems.

Csomagtároló a Rovás Linux eszközkészlet telepítéséhez és frissítéséhez Ubuntu, Xubuntu, Linux Mint és kompatibilis Debian-alapú rendszereken.

What this repository provides

This APT repository contains the Rovas desktop packages, fonts, keyboard layout, Hungarian locale support, transliterated message catalogs, browser support, wallpapers, and the Rovas Tools control panel.

The easiest install path is the script below. It adds the signing key, configures the APT source, refreshes package metadata, and installs rovas-desktop.

curl -fsSL https://nocodesolutions.github.io/rovas-apt/install.sh | sudo bash

Mit ad ez a tároló

Ez az APT tároló tartalmazza a Rovás asztali csomagokat, betűkészleteket, billentyűzetkiosztást, magyar locale támogatást, átírt üzenetkatalógusokat, böngészőtámogatást, háttérképeket és a Rovás Eszközök vezérlőpanelt.

A legegyszerűbb telepítés az alábbi paranccsal történik. Hozzáadja az aláírókulcsot, beállítja az APT forrást, frissíti a csomaglistát, majd telepíti a rovas-desktop csomagot.

curl -fsSL https://nocodesolutions.github.io/rovas-apt/install.sh | sudo bash

Manual APT source

If you prefer to inspect the setup manually, use this source entry with the repository keyring at /usr/share/keyrings/rovas-tools-archive-keyring.gpg.

deb [arch=all signed-by=/usr/share/keyrings/rovas-tools-archive-keyring.gpg] https://nocodesolutions.github.io/rovas-apt stable main

Published packages

  • rovas-desktop and rovas-full for complete installs
  • rovas-tools for the control panel and command line utility
  • fonts-rovas, rovas-keyboard, rovas-locale-hu, and rovas-l10n-hu for core language support
  • rovas-browser-chromium and rovas-wallpapers for optional desktop integration

Kézi APT forrás

Ha kézzel szeretné ellenőrizni a beállítást, ezt a forrássort használja a /usr/share/keyrings/rovas-tools-archive-keyring.gpg kulcstárral.

deb [arch=all signed-by=/usr/share/keyrings/rovas-tools-archive-keyring.gpg] https://nocodesolutions.github.io/rovas-apt stable main

Közzétett csomagok

  • fonts-rovas, rovas-keyboard, rovas-locale-hu és rovas-l10n-hu az alap nyelvi támogatáshoz
  • rovas-browser-chromium és rovas-wallpapers opcionális asztali integrációhoz
